cl_number- constructor 1.0
--------------------------

Side project to the next class- usage project.

The constructor of the class which assigns the data fields of the class with the numbers given as the argument when creating an object of the class.

The usage is:

  cl_number object (number_1, number_2);
  
Two numbers can be given to the object which are stored inside.
